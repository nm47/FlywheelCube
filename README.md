# Flywheel Cube
 A reaction wheel LQR balanced cube using the teensy 3.5

![Build Status][travis-image]


#### Note: This project is a work in progress, the current status can be found under "Roadmap".

## Installation
Simply Download the zip and run, PID/Kalman tuning parameters can be found in the Cube.cpp file.

Note: This project was designed to run on the Teensy3.5. Other versions of the teensy will require slight wiring modifications as the 3.5 is 5v tolerant on most pins and others are not. 

TeensyDuino must be installed to program the teensy through the arduino IDE.
TeensyDuino can be found here:

https://www.pjrc.com/teensy/td_download.html
